## 快速了解

**告警视听助手** 是一款面向运维监控场景的 **Python + Flet** 桌面工具：

- 从监控平台拉取 **历史 / 活跃告警**，定时轮询并 Diff 新增与恢复事件  
- 按 **文本规则** 生成播报话术，按 **播报策略** 控制重复次数（UI 已支持，部分后端待接）  
- 支持 **Windows SAPI**、**MOSS-TTS-Nano 本地**（CPU/GPU）、**讯飞在线** 三种 TTS  
- **MOSS** 通过 `moss_daemon.py` 常驻加载模型；可选 **片段缓存 + 拼合** 降低延迟（方案已写，实现进行中）

| 项目 | 说明 |
|------|------|
| 技术栈 | Python 3.9（主程序）+ Python 3.12（MOSS）、Flet 0.28 |
| 运行平台 | Windows（SAPI / MOSS ONNX） |
| UI 入口 | `flet_demo.py` |
| 启动命令 | `.\run.ps1` |
| 运行日志 | `logs/app.log`、`logs/moss_daemon.log` |
