# 使用指南

本文档说明如何安装、配置和日常使用告警视听助手。

---

## 一、环境要求

| 项目 | 要求 |
|------|------|
| 操作系统 | Windows（本地 TTS 依赖 Windows SAPI） |
| Python | 3.8 及以上 |
| 网络 | 访问监控平台 API；使用讯飞 TTS 时需联网 |

---

## 二、安装步骤

请在项目根目录 `alarm_speech` 下执行以下命令。

### 2.1 克隆仓库

```bash
git clone https://github.com/HUANGYIHA0382/alarm_speech.git
cd alarm_speech
```

### 2.2 创建虚拟环境（推荐）

```bash
python -m venv alarm_env
alarm_env\Scripts\activate
```

### 2.3 安装依赖

```bash
pip install -r requirements.txt
```

主要依赖：

- `flet`：桌面 UI 框架
- `requests`：HTTP 请求
- `websocket-client`：讯飞 TTS WebSocket 通信

如需更稳定的 Windows SAPI 支持，可额外安装：

```bash
pip install pywin32 pyttsx3
```

## 三、配置说明

### 3.1 监控平台连接（必填）

启动程序后，在界面左侧填写：

1. **服务器 HOST**：监控平台地址，如 `http://192.168.1.100:8080`
2. **访问令牌 Token**：平台颁发的 API Token

这两项不会保存到代码仓库，每次启动需在界面填写（或后续可自行扩展持久化）。

### 3.2 讯飞 TTS（可选）

若希望使用在线语音而非 Windows 本地语音，在项目根目录创建 `config.local.json`：

```json
{
  "tts": {
    "provider": "xfyun",
    "xfyun": {
      "app_id": "你的APPID",
      "api_key": "你的APIKey",
      "api_secret": "你的APISecret",
      "vcn": "xiaoyan"
    }
  }
}
```

也可通过环境变量配置：

```bash
set XFYUN_APP_ID=你的APPID
set XFYUN_API_KEY=你的APIKey
set XFYUN_API_SECRET=你的APISecret
```

> `config.local.json` 已在 `.gitignore` 中忽略，请勿将密钥提交到 Git。

未配置讯飞时，程序自动使用 Windows SAPI 离线语音。

## 五、日常操作流程

推荐按以下顺序使用：

### 步骤 1：填写连接信息

在左侧面板输入 **HOST** 和 **Token**。

### 步骤 2：拉取历史（可选）

点击 **「历史告警 (100条)」**，确认能正常连接并查看告警格式。右侧终端会显示拉取结果。

### 步骤 3：配置语音

- 选择 **播报规则**（一般用「自动匹配」即可）
- 选择 **语音引擎**（离线 SAPI 或讯飞在线）
- 可点击 **「试听 SAPI」** 或 **「测试语音」** 验证播报效果
- 确认 **「语音播报」** 开关已打开

### 步骤 4：开始轮询

1. 选择轮询间隔：**30 秒** 或 **1 分钟**
2. 点击 **「开始」** 进入自动轮询
3. 有新告警或设备恢复时，终端会记录事件并自动语音播报
4. 需要暂停时点击 **「停止」**

### 步骤 5：观察终端

右侧「数据终端」会显示：

- 事件播报日志（新增、恢复等）
- 当前活跃告警列表
- 语音测试的详细转换结果

## 六、常见问题

### Q1：提示「HOST 或 Token 为空」

请在左侧面板完整填写服务器地址和 Token 后再操作。

### Q2：轮询失败 / HTTP 错误

- 检查 HOST 地址是否正确（含 `http://` 或 `https://`）
- 检查 Token 是否有效、是否过期
- 确认本机能访问监控平台网络

### Q3：讯飞引擎切换失败

- 确认 `config.local.json` 中 `app_id`、`api_key`、`api_secret` 已正确填写
- 确认网络可访问讯飞 TTS 服务
- 可改用「Windows SAPI 离线」引擎

### Q4：本地语音无声或发音奇怪

- 在 Windows「设置 → 时间和语言 → 语音」中安装中文语音包
- 使用「试听 SAPI」测试，并调节语速滑块
- 端口表类告警建议使用标准话术格式（见 [功能说明](./功能说明.md)）

### Q5：启动轮询后没有立即播报

首轮轮询仅建立追踪基线，不会对已有告警播报，这是正常设计。之后出现 **新增** 或 **恢复** 事件才会播报。
